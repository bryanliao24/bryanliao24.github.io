Projects
Full-stack Visualization of Analyzing NBA Shooting Performance

• Developed a responsiveness web app with React and Flask to provide advanced visual analytics of NBA_API data, allowing users to compare players across different scenarios.
• Implemented complex visualization expression including Shot, Cell, Zone, and Region with D3.js while enhancing user interaction with zoom and hover features.
• Designed and integrated a robust filtering system to facilitate in-depth analysis of player performance, enabling customized comparisons based on different players.

Hybrid Movie Recommendation System

• Developed a hybrid recommendation system across diverse use cases integrating Matrix Factorization, KNN and DNN with MovieLens 10M dataset ensuring robust and personalized recommendations in real-world applications.
• Implemented KNN with TF-IDF vectors generated from movie overviews to address cold start problems and enable content-based recommendations and optimized by cosine similarity.

Covid-19 Prevention Health Monitor App

• To prevent Covid-19, developed an Android application to collect data related to human health, with a variety of health data, including heart rate and respiration and so on.
• Algorithms such as heart rate are used using multiple Android sensors including acceleration sensor values and rear camera detection and calculating the peak red pixel change in the area to determine the heart rate.
• Use SQLite to create a lightweight database to store body data .
